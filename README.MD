# lyre

使用 AI 处理的 RIME 音乐词库

```text


      ||             ||
    @@##===#######===##@@
      ||   |||||||   ||
      ||   |||||||   ||
      \\   |||||||   //
       \\__|||||||__//
        \  |||||||  /
         \_#######_/


```

本词库以音乐学术著作与专业教材为语料来源，使用 Deepseek R1、Qwen QWQ 进行分词。

词库内容涵盖音乐理论术语、乐器名称、作曲家、经典作品、演奏技法等专业领域词汇。

## 文件说明

- `lyre.dict.yaml` ： 混合了所有词库的文件
  - `electronic.lyre.dict.yaml` ： 电子音乐
  - `classical.lyre.dict.yaml` ： 古典
  - `mix.lyre.dict.yaml` ：混音、后期处理
